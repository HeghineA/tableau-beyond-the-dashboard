02 Tableau as a Visualisation Tool

Charts are not decoration. They encode meaning.

Marks Card

ColorSizeLabelDetailTooltipShape

Each one is a channel that carries data into the eye.

The Marks Card is where Tableau turns a column of numbers into something the brain can read in one second.

Chart choice = question type

Bar → comparison ("which is biggest?")

Line → trend over time ("is it rising?")

Map → location ("where is it happening?")

Scatter → relationship ("does X track with Y?")

Most dashboards
fail here.

They show data but never tell you what to do about it.

Cherry Blossom Technical Layer

The 7 calculations that power the Sakura Blooming dashboard.

FBD (Full Bloom day-of-year)

DATEPART('dayofyear', [Full Bloom Date])

Bloom Duration

DATEDIFF('day', [First Bloom Date], [Full Bloom Date])

Regions (102 cities → 9 regions)

CASE [Site Name] WHEN "Sapporo" THEN "Hokkaido" WHEN "Sendai" THEN "Tohoku" WHEN "Tokyo" THEN "Kanto" WHEN "Kyoto" THEN "Kansai" WHEN "Hiroshima"THEN "Chugoku" WHEN "Fukuoka" THEN "Kyushu" … /* 90+ more cities */ ELSE "Other / Check Site Name" END

Day Diff (yearly bloom spread)

{ FIXED [Year] : MAX([FBD]) } - { FIXED [Year] : MIN([FBD]) } // → range of bloom days within each year

Today (this month)

MONTH(TODAY())

Bloom Allignment (blooming now)

[Today] = MONTH([First Bloom Date]) // → flags regions blooming THIS month

Global Warming (post-2005 era)

IF YEAR([Year]) >= 2005 THEN "Modern Warming Period" END // → climate-shift overlay on Blooming Trend

COVID Period (period bucket)

IF [flt_date] < #2020-03-01# THEN "Before COVID" ELSEIF [flt_date] <= #2021-12-31# THEN "During COVID" ELSE "After COVID" END

Recovery KPI (LOD ratio)

{ SUM(IF YEAR([flt_date]) = 2025 THEN [Flights Total] END) / SUM(IF YEAR([flt_date]) = 2019 THEN [Flights Total] END) }

COVID Drop (2020 vs 2019)

( SUM(IF YEAR([flt_date])=2020 THEN [Flights Total] END) - SUM(IF YEAR([flt_date])=2019 THEN [Flights Total] END) ) / SUM(IF YEAR([flt_date])=2019 THEN [Flights Total] END)

Airports per State (FIXED LOD)

{ FIXED [state_name] : COUNTD( IF YEAR([flt_date]) = 2020 THEN [apt_icao] END ) }

2016–2019 baseline (window avg)

WINDOW_AVG( IF YEAR(MIN([flt_date])) < 2020 THEN SUM([Flights Total]) END )

Deviation vs baseline

( SUM([Flights Total]) - [2016-2019 AVG] ) / [2016-2019 AVG]

Two Dashboards, Two Mindsets

Same tool. Two completely different questions.

Cherry Blossom

"How is seasonal timing changing and where should I go to see it?"

Logic: day-of-year, bloom duration, region grouping, trend smoothing, current-month alignment.

Mood: emotional, cultural, exploratory.

Airport Traffic

"Where did traffic fall and how unevenly is it recovering?"

Logic: period classification, 2019 baseline, drop %, recovery ratio, FIXED LODs, deviation.

Mood: operational, decision-driving, accountability.

Same skeleton: question → logic → encoding → action. The skill is recognising which question you’re answering.
